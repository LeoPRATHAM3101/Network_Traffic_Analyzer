# 🌐 Intelligent Network Traffic Analyzer

## 📌 Overview
This project performs advanced network traffic analysis using Wireshark and Python. It analyzes captured packets to identify protocols, detect suspicious activities such as port scanning, and highlight potential security threats.

---

## 🚀 Features

✔ Packet analysis from `.pcap` files  
✔ Protocol identification (TCP, UDP)  
✔ Detection of most active IP address  
✔ Suspicious port monitoring (SSH, FTP, RDP, SMB)  
✔ Port scanning detection 🚨  
✔ Automated analysis using Python (Scapy)  

---

## ⚙️ How It Works

1. Capture network traffic using Wireshark  
2. Save the capture as `.pcap` file  
3. Run Python script to analyze packets  
4. Detect suspicious patterns and generate output
